# Create a new WinForms application

```
dotnet new winforms -n MyWinFormsApp
```

# Create a new WPF application

```
dotnet new wpf -n MyWpfApp
```

# Navigate to the project directory

```
cd MyWinFormsApp
```

# Build the project

```
dotnet build
```

# Run the application

```
dotnet run
```

# List all available project templates

```
dotnet new --list
```
