JavaScript Variables and Scopes

Variables

In JavaScript, we have three ways to declare variables:

  • var: Function-scoped or globally-scoped
  • let: Block-scoped
  • const: Block-scoped, cannot be reassigned

Scopes

JavaScript has several types of scopes:

  • Global Scope: Variables accessible from anywhere
  • Function Scope: Variables accessible only within a function
  • Block Scope: Variables accessible only within a block (introduced with ES6)

Example


// Global scope
var globalVar = "I'm global";

function exampleFunction() {
    // Function scope
    var functionVar = "I'm function-scoped";
