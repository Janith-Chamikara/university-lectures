Topic: Cascading style sheets(CSS)

Date: 2024-10-29

Key Points:

  • Ways of integrating CSS into html
  • CSS priority

Summary:

There are three main ways to add CSS to your HTML. First, there’s inline CSS, where you add styles directly to a specific element using the style attribute—perfect for small, quick adjustments. Next, there’s internal CSS, which goes inside a <style> tag in the <head> of your HTML. This is useful for styling a single page. Finally, external CSS is when you put your styles in a separate file and link it to your HTML with a <link> tag. This keeps things organized and is great for making sure multiple pages all use the same styles.

External CSS file
Embedding CSS internally using style tag
        <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Document</title>
            <style>
              h1 {
                text-align: center;
              }
              h2 {
                color: green;
              }
            </style>
        </head>
Inline CSS styles
        <h1 style="color:red;padding-left:0px;">Hello madam</>

CSS styles can also be local or global. Local styles apply only to specific sections or pages, like with inline or internal CSS. Global styles, usually in an external CSS file, apply across the whole site to keep everything consistent.

When it comes to CSS priority, some styles have more weight than others. Inline styles have the highest priority, followed by internal, and then external styles. There’s also a rule for specificity: IDs take priority over classes, and classes have more weight than basic tags like <p> or <h1>. This order of priority makes sure that the most specific styles are applied first.
